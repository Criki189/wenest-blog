# Keyword performance log

Track which keywords we targeted, where they ranked, and how much organic traffic they brought.

## How to use

1. Update **monthly** (first Monday of the month).
2. Pull data from Google Search Console + Ahrefs free tools.
3. Star (★) any keyword in top 10 — those are the cluster anchors. Build more articles around them.
4. Cross out (~~strikethrough~~) keywords stuck below position 50 after 90 days. They are not winnable for us; archive the article or rewrite.
