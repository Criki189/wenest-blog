# Voice refinements

A running log of concrete voice corrections from human review. The system prompt is updated periodically based on patterns here.

## How to use

When editing a Claude-generated article, if you change wording for **brand voice** reasons (not just typo or fact correction), log it here. Capture:

- What Claude wrote (red flag).
- What we changed it to (correct voice).
- Why (the principle).

Patterns that show up 3+ times → promote to `CLAUDE.md` §2 and to the system prompt.
